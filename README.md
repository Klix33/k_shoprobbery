# k_shoprobbery
[FREE] [ESX] Shop Robbery:
- Prosty i przejrzysty kod
- Prosty Config
- Możliwość ustawienia dowolnych miejsc na napad
- Możliwość ustawienia wysokości łupu i cooldownu między napadami
- Musi być minimum 1 osoba z Policji

Wymagania:
- ESX Framework
- ox_lib
- ox_target
